Tailstrap

A collection of Bootstrap-inspired components written in Tailwind.

Migrating from Bootstrap to Tailwind

Tailwind and Bootstrap are very different frameworks. Bootstrap makes it much easier to get started, but makes it much harder to customize your project.

By starting with this set of Tailwind-style components, you get the benefit of starting quickly and the benefit of easy customizability.

Contributing

This project is open source, and is currently missing many components. Contribute at the Tailstrap Github page to earn some sweet imaginary internet points.
